# 7-Day Nextjs Web Design

A seven-day design workflow for web products that will be built in Next.js.

Key files:

- `skill.json`
- `core/skill-definition.md`
- `platforms/claude/.claude/skills/7-day-nextjs-web-design/SKILL.md`
- `platforms/codex/plugins/7-day-nextjs-web-design/.codex-plugin/plugin.json`
- `platforms/cursor/.cursor/rules/7-day-nextjs-web-design.mdc`

This skill keeps the same seven-day cadence as the generalized web design skill, but its outputs are tailored for Next.js implementation planning and developer handoff.
